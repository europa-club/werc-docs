# Contributing

Thank you for your interest in contributing.

## Scope
This repository hosts public documentation for Euro Finance (WERC), including:
- official links and disclosures
- networks and contract references
- security and reporting information

## How to propose a change
1. Open an Issue describing the change (optional but preferred for non-trivial edits).
2. Create a Pull Request with a clear title and a short summary.
3. Provide references (explorer links, official website pages) where applicable.

## Content rules
- Keep disclosures clear and prominent (Not Circle / Not EURC).
- Avoid third-party copyrighted images or logos unless you have permission.
- Use verifiable links for contracts and explorers.

## Security
For vulnerability reports, see SECURITY.md.
